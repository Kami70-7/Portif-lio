Eu sou um gato

O projeto é uma página web criada com HTML e CSS, destinada para o gato Jorjinho, suas irmãs, e sua humana. O objetivo é criar um ambiente onde pessoas que gostam de animais possam ver o gato Jorjinho e sua família.

A estrutura do site é baseada em HTML, que define a semântica e a layout do conteúdo, enquanto o CSS é usado para estilizar a página de maneira visual

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site. Cada ser que faz parte da vida de Jorde é destacado em seu próprio contêiner. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária.
